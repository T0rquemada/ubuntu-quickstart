# Ubuntu Personal Quickstart

## Quickstart
```sh
git clone https://github.com/T0rquemada/ubuntu-personal-quickstart.git
```
```sh
cd ubuntu-personal-quickstart
```
```sh
chmod +x ./ubuntu.sh
```
```sh
./ubuntu.sh
```
