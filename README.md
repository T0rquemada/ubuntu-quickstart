# Ubuntu Personal Quickstart

## Quickstart
```sh
git clone https://github.com/T0rquemada/ubuntu-personal-quickstart.git
```
```sh
cd ubuntu-personal-quickstart
```
```sh
chmod +x ./ubuntu.sh
```
```sh
./ubuntu.sh
```

## Installs:
### Dev packages: 
- Python (apt)
- C/C++ (apt)

### Programs:
Dev:
- VScode / Zed / Both

Utils:
- Gnome Boxes (apt)
- Libreoffice Calc & Writer (apt)
- Ollama

Personal:
- Obsidian (flatpak)
- Bitwarden (flatpak)

Other:
- qBittorent (flatpak)
- FreeCad (flatpak)

Communication:
- Signal (flatpak)
- Discord (flatpak)
