# Ubuntu Personal Quickstart

## Quickstart
```sh
git clone https://github.com/T0rquemada/ubuntu-personal-quickstart.git
```
```sh
cd ubuntu-personal-quickstart
```
```sh
chmod +x ./ubuntu.sh
```
```sh
./ubuntu.sh
```

## Installs:
### Dev packages: 
- Python (apt)
- C/C++ (apt)
- PHP (apt)

### Programs:
Dev:
- VScode / Zed / Windsurf

Utils:
- Gnome Boxes (apt)
- Libreoffice Calc & Writer (apt)

Personal:
- Obsidian (flatpak)
- Bitwarden (flatpak)

Other:
- qBittorent (flatpak)
- FreeCad (flatpak)

Communication:
- Signal
- Discord (flatpak)
